# StuartHua's Blog

> Personal Blog

* [https://blog.stuarthua.com/](https://blog.stuarthua.com/)

## Init

```shell
hugo mod init my-site.local
hugo mod tidy
```

## Usage

update theme - [hugo-theme-puff](https://github.com/stuarthua/hugo-theme-puff):

```shell
hugo mod get -u
```

run server:

```shell
hugo server -D
```
